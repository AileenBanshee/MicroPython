# MicroPython
Creating things using Micropython on Arduino Nano RP2040 Connect

Machine and tools:
- Arduino Nano RP2040 Connect
- MicroPython language
- Seed Pir Motion Sensor
- NeoPixel LED Ring with 24 x RGB LEDs
- Big Sound Sensor Module
- Arduino Servo Motor
- Grove - Touch Sensor
- Grove - Water Sensor
